1. 健診の種類と全体像
初めての方へ
倉敷市の健診は大きく「生活習慣病予防健診」と「がん検診」に分かれます。受診者はすべて「けんしん受診券」を持参して来院します。受診券の内容を確認することが受付業務の基本です。

4. 受診券の見方

受診券の各欄の意味

欄名内容・確認ポイント
①検診者番号欄 10桁の番号。女性の一般健診・がん検診の検診票に転記する。転記後に必ず再確認。
②費用区分欄（がん検診等） 「一般」「減免」「高齢」のいずれかが印字。
減免された受診券は、結果報告書の費用区分も「減免」で記入。
③料金欄 受診可能 → 自己負担額が数字で印字。
受診不可 → 「＊＊＊＊＊＊」と表示（受診させない）。
④受診欄 実施した健診部分に医療機関名または日付を記入（ゴム印可）し、受診者に受診券を返却する。
⑤肝炎ウイルス検診欄 B型のみ・C型のみ・両方の確認に使う（前掲の表参照）。
⑥回収が必要な部分 胸部レントゲン検診・生活習慣病予防健診を実施した場合は、それぞれの部分を切り取り回収する。
⑦受診券整理番号欄 11桁の番号。生活習慣病予防健診（A・B・C）の健診票に転記する。
⑧費用区分欄（生活習慣病予防健診） 「一般」「減免」「高齢」のいずれかが印字。
⑨報告区分欄 「A」「B」「C」「AC」のいずれかが印字。詳細は第6章参照。
絶対に確認すること
受診券の料金欄が「＊＊＊＊＊＊」の検診は対象外です。受診させないでください。また、今年度の受診券を必ず確認してください（昨年度のものは無効）。
5. 受付時の手順（チェックリスト）

受付の流れ

受診券の提出を求める
→
今年度の受診券か確認
→
料金欄で受診可否・金額確認
→
保険資格確認（A/B）
→
自己負担金の徴収
→
健診票に番号転記・再確認

受付チェックリスト

  • 今年度のけんしん受診券を持参しているか確認した
  • 料金欄が「＊＊＊＊＊＊」でないか確認した（対象外検診でないか）
  • 国保特定健診（A）または長寿健診（B）の場合、マイナ保険証（資格確認書）で当日の保険資格を確認した
  • 受診券の料金欄に記載の金額を自己負担金として徴収した（クーポン・節目無料は0円）
  • 健診票の受診券整理番号または検診者番号を正確に転記し、転記後に再確認した
  • 複写式健診票はボールペンで強く記入するよう受診者に促し、2・3枚目まで鮮明に写っているか確認した
  • 夫婦や家族で来院した場合、番号の入れ替わりに特に注意した
  • 生活習慣病予防健診・胸部レントゲン検診を実施した場合、受診券の該当部分を切り取り回収した
  • 他の検診は受診券に医療機関名または日付を記入して返却した

減免・受診券再交付について

  • 料金の減額を希望する場合は、受診日前に申請が必要です。受診後の減額対応はできません。
  • 受診券を紛失した患者さんが来た場合：受診させないこと。まず受診券の再交付手続きを案内してください。
  • 減免・再交付の申請先：最寄りの保健推進室または倉敷市電子申請サービス

結果通知について

  • 結果通知は実施要領に基づき、時期を逸することなく速やかに行ってください。
  • 郵送による結果通知を活用してください（郵送料は委託料に含まれています）。
  • 要精密検査の場合は精密検査依頼書を同封して通知します。
  • 肝炎ウイルス検診で陽性の場合：検診票3枚目（肝・様式1号③）を同封して通知します。
8. よくある誤り・Q&A

過去に実際にあった誤り事例

誤り正しい対応
昨年度のクーポン券を持参した市民に今年度の検診を無料で受診させた 必ず今年度の受診券で確認する。昨年度のクーポンは無効。
肝炎ウイルス検診の節目無料対象者から料金を徴収した 受診券の肝炎欄「今回は無料」の記載を確認する。費用区分は「減免」で報告。
受診券を持参していない市民を受診させた（→重複受診になった） 受診券なしでは受診させない。再交付の手続きを案内する。
夫婦の検診票に相手の検診者番号を記入してしまった（番号の入れ替わり） 番号転記後に必ず再確認。家族来院時は特に注意。
要精密検査の結果通知をせずに3か月以上が経過した 時期を逸せず速やかに郵送通知する。
クーポン・節目無料対象者の費用区分を「無料」と記載した 費用区分は「減免」で記載する。
複写が薄くて読めない健診票を提出した 記入後に2・3枚目まで確認。薄い場合は医療機関側で追記して判読可能にする。

Q&A

QA
受診勧奨ハガキ（SMS）だけを持ってきた方は受診できますか？ できません。受診勧奨通知に受診券はついていません。健康づくり課けんしん係（086-434-9866）への連絡を案内してください。
受診券を紛失したとのことで受診を希望している場合は？ 受診させないこと。再交付手続きを案内します。再交付申請は電子申請・電話でも可能です。
胃がん検診の患者さんで生検を実施した場合は？ 胃がん疑いによる生検の場合は判定コード「02」・指示「要精密検査」とします。生検結果を精密検査結果通知書に記入して岡山県がん登録室へ送付。
大腸がん検診で「擬陽性」の場合は？ 一つでも「陽性」または「擬陽性」なら指示を「要精密検査」としてください。
心電図で心房細動がみられた場合は？ 電子報告の場合は心電図所見欄に「所見あり」を選択し「心房細動」と入力。受診者への指導と情報提供リーフレット配付が必要。
帳票が不足してしまったら？ 事前連絡の上、最寄りの保健推進室または健康づくり課けんしん係に取りに来てください。

10. 各がん検診・特殊検診の詳細

胃がん検診（内視鏡 または 胃部X線）

項目内容
対象者50歳以上の市民（2年に1回）
受診不可妊婦・妊娠疑い・内視鏡で経過観察中の方
検査項目胃内視鏡検査 または 胃部エックス線検査（どちらか選択）
受診前に患者さんへ必ず伝えること（前日〜当日）
  • 前日：夕食は消化の良いものにし、21時までに済ませる。以降は飲食しない（水・お茶・就寝前の薬はOK）
  • 当日：飲食禁止・タバコは控える
  • 常用薬：高血圧薬等は主治医に相談の上、3時間前までにコップ1杯の水で服用可
  • 糖尿病治療中の方：主治医の指示のもと、朝の内服薬・インスリン注射は控える（低血糖防止）
バリウム検査（胃X線）特有の注意点
  • 検査後は下剤を服用し、水分をしっかり摂る
  • 翌日夕方までにバリウムが出ない場合は医師へ相談
  • むせやすい方・高齢者は誤嚥リスクあり。必要に応じて事前相談を促す
胃内視鏡検査 特有の注意点
  • 検査後の飲食・車の運転等は医療機関の指示に従う
  • 生検（組織採取）を行った場合は検診対象外→医療保険適用で自己負担金が発生

保管方法1回目の検体は冷蔵保存（冷蔵庫推奨）
判定結果陰性 / 疑陽性 / 陽性 / 判定不能
疑陽性・陽性のいずれかが出たら 「要精密検査」

肝炎ウイルス検診

項目内容
対象者（A）当該年度に 40歳になる市民
対象者（B）41歳以上の市民で 過去に一度も受診したことがない 方
検査項目問診 ＋ HBs抗原検査（B型） ＋ HCV抗体検査（C型）
HCV追加検査HCV抗体が中力価・低力価の場合は HCV核酸増幅検査 を追加実施
重要この検診は 原則一生に1回のみ（同じ種類は2回目以降受診不可）
受診券確認ポイント：「B型のみ」「C型のみ」の印字がある場合、過去に一方を受診済みです。印字がない場合はB型・C型両方を実施します。
陽性判定の場合：肝炎一次専門医療機関への受診を勧奨してください（速やかに）。

前立腺がん検診（PSA検査）

項目内容
対象者50歳以上の市民（男性）
受診不可前立腺がん治療中・前立腺全摘手術を受けた方（経過観察中も含む）
検査項目問診 ＋ 採血による PSA（前立腺特異抗原）検査
PSA要精密検査の基準値（令和8年度）
年齢（年度末）基準値
50〜64歳3.0 ng/ml 以上
65〜69歳3.5 ng/ml 以上
70歳以上4.0 ng/ml 以上
薬の影響に注意：男性型脱毛症の治療薬（フィナステリド等）や前立腺肥大症の治療薬を服用中の方はPSA値が低く検出される場合があります。泌尿器科専門医や主治医への相談を促してください。

胸部レントゲン検診（肺がん・結核）

項目内容
対象者40歳以上の市民（妊婦・妊娠疑いは除く）
検査項目（全員）質問（問診） ＋ 胸部エックス線検査
喀痰細胞診（追加）50歳以上 かつ 喫煙指数600以上（過去の喫煙者含む）の方に実施
喫煙指数 ＝ 1日の喫煙本数 × 喫煙年数（加熱式タバコはカートリッジ本数で換算）
受診券の取扱い胸部レントゲン検診部分を回収（返却不要）
血疾の訴えがある場合検診ではなく有症状の疑いとして、専門医療機関への受診を案内
受診券は切り取って回収してください。他の検診（がん検診等）は受診券に日付を記入して返却しますが、胸部レントゲン検診のみ回収です。

女性の一般健康診査

項目内容
対象者20〜39歳の市民（女性）
検査項目 問診・身体計測（身長・体重・BMI）・理学的検査・血圧測定・尿検査（尿糖・尿蛋白・尿潜血）
血液検査：LDL・HDL・中性脂肪・赤血球数・血色素量（Hb）・ヘマトクリット
注意点国保特定健診・長寿健診の対象外となる年代の女性向け健診
