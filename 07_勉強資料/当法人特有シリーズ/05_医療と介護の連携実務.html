1. 連携の全体像

いなだ医院を中心とした医療と介護の連携は、患者さんが安心して自宅や施設で生活し続けるために欠かせない仕組みです。医師・看護師・医療事務スタッフがそれぞれの役割を果たすことで、医療と介護のサービスがスムーズにつながります。

医療事務スタッフの役割

医療と介護の連携においては、医療事務スタッフが最初の窓口となる場面が多くあります。ケアマネジャーや訪問看護ステーションからの電話・FAX・来院対応が中心です。

  • ケアマネジャーからの書類依頼（主治医意見書・診療情報提供書）の受付
  • 訪問看護ステーションからの指示書依頼の受付
  • 市区町村からの意見書依頼文書の受取・管理
  • 作成された書類の封入・送付・FAX送信
  • 介護保険関連書類の保管・ファイリング
ポイント：書類の依頼が来たら、まず「誰から・何の書類・いつまでに必要か」を確認することが大切です。期限が短い場合は医師に早めに伝えましょう。

2. 診療情報提供書（紹介状）

📄 診療情報提供書とは

医師が、他の医療機関や介護事業所（ケアマネジャーなど）に対して、患者さんの病状・治療内容・生活上の注意点などを文書でお伝えするものです。「紹介状」と呼ばれることもありますが、介護の場面では情報提供が目的の場合も多くあります。

💴 診療報酬（点数）
区分点数算定条件
診療情報提供料（Ⅰ）250点月1回まで算定可

※ 250点 = 2,500円の医療費が医療機関に支払われます（保険者負担分含む）

📋 ケアマネジャー宛に出す主なケース
  • 居宅療養管理指導を開始・継続する際に病状・注意点を伝えるとき
  • 患者さんの状態が変化し、ケアプランの見直しが必要と医師が判断したとき
  • 退院後の在宅生活に向け、サービス内容の調整をケアマネに依頼するとき
  • 施設入所・転院の際に情報を引き継ぐとき

3. 主治医意見書

📄 主治医意見書とは

患者さんが要介護認定（介護保険を利用するための認定）を申請する際に、市区町村が主治医に作成を依頼する公的文書です。患者さんの心身の状態・病名・医療処置の内容・生活機能の状況などを記載します。介護認定審査会での審査資料として重要な役割を担います。

💴 費用（謝礼金）

主治医意見書は医療保険の診療報酬ではなく、市区町村から医院に謝礼金が支払われます。金額は地域によって異なります。

4. 居宅療養管理指導

📄 居宅療養管理指導とは

医師・歯科医師・薬剤師・管理栄養士などの専門職が、通院が困難な患者さんの自宅（居宅）を訪問し、療養上の管理・指導を行う介護保険サービスです。医療保険の訪問診療とは別の保険（介護保険）で請求する点が特徴です。

5. ケアマネジャーとのやり取りで多い問い合わせ

ケアマネジャーから医院へ連絡が入る際、医療事務スタッフが一次対応をする場面があります。よくある問い合わせパターンと対応の基本を確認しておきましょう。

📞 「主治医意見書をお願いしたい」

依頼の背景：患者さんが要介護認定を申請または更新する際に、ケアマネジャー（または患者家族）から連絡が入ることがあります。

対応の基本手順：

  1. 患者氏名・生年月日・連絡先を確認する
  2. 市区町村から依頼書（様式）が届いているか確認する
  3. 期限を確認し、医師に作成依頼を伝える
  4. 作成後、事務スタッフが市区町村へ発送する

【要確認：当院での対応手順】

📞 「情報提供書（診療情報提供書）をお願いしたい」

依頼の背景：ケアプランの作成・見直し、または新しいサービス導入にあたり、患者さんの医療情報が必要な場合に依頼が来ます。

対応の基本手順：

  1. 患者氏名・依頼の目的・提出先・期限を確認する
  2. 医師に依頼内容を伝え、作成を依頼する
  3. 作成された書類を確認し、封入またはFAX送信する
  4. 送付完了をケアマネに連絡する（必要に応じて）

【要確認：当院での対応手順】

📞 「患者さんの薬の内容を教えてほしい」

依頼の背景：ケアプランや食事調整のために、服薬情報の確認を求められることがあります。

個人情報の観点から要注意：薬の情報は個人情報です。電話口でそのままお伝えすることは原則できません。医師または薬剤師に確認が必要であることをお伝えし、対応方法を仰ぎましょう。患者さんご本人またはご家族の同意がある場合は、医師の判断で情報提供が可能なことがあります。

【要確認：当院での対応手順】

📞 「訪問看護の指示書をお願いしたい」

依頼の背景：訪問看護ステーションが看護サービスを提供するためには、主治医が発行する「訪問看護指示書」が必要です。新規・更新の際に依頼が来ます。

対応の基本手順：

  1. 依頼元（訪問看護ステーション名・担当者名・連絡先）を確認する
  2. 患者氏名・指示期間（通常1〜6か月）・特記事項を確認する
  3. 医師に作成を依頼する
  4. 作成後、訪問看護ステーションに郵送またはFAX送信する
訪問看護指示書の有効期間は原則6か月以内。更新時期が近づいたら、先方から連絡が来る前に確認できると丁寧な対応です。

【要確認：当院での対応手順】

7. まとめ

🔗
連携の窓口は事務スタッフ
書類依頼・情報共有の多くは医療事務スタッフが最初に対応します。「誰から・何の目的で・いつまでに」を必ず確認しましょう。
📄
書類の種類を覚える
主治医意見書・診療情報提供書・訪問看護指示書・居宅療養管理指導の4つが特に頻度が高いです。依頼元と提出先を混同しないよう注意しましょう。
💴
保険の種類に注意
医療保険（診療報酬）と介護保険（介護給付費）は請求先が異なります。居宅療養管理指導は介護保険（国保連）への請求になります。
🔒
個人情報の取り扱い
薬の内容・病状など患者さんの個人情報は、本人・家族の同意なく第三者に伝えることはできません。判断に迷ったら必ず医師に確認しましょう。
