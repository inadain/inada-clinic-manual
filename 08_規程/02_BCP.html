いなだ医院　事業継続計画（BCP）

制定：2026年4月14日　最終改定：2026年6月　適用：医療法人いなだ医院　全職員

第1章　総則

1.1　目的

本計画は、大規模地震・水害などの自然災害、新興感染症のパンデミック、および電子カルテのダウン・サイバー攻撃などのシステム障害によって診療業務へ重大な影響が生じた際に、以下を達成するために定める。

  • 患者および全スタッフ（医師2名・看護師12名・医療事務8名、計22名）の生命と安全を最優先に確保する
  • 内科・小児科かかりつけ患者への投薬・処方、および訪問診療患者のケアを可能な限り継続する
  • 地域の医療提供体制の崩壊を防ぎ、地域住民の生命と健康を守る

1.2　基本方針

  • 人命に危険が及ぶ状況下では、避難を診療継続より優先する
  • 診療の中断・縮小はあっても、完全停止からの最速復旧を目指す
  • 玉島医師会・倉敷中央病院をはじめとする地域医療機関・自治体と密接に連携する
  • 全職員が本BCPを理解し、年1回以上の訓練と定期見直し（BCM）を継続する

1.3　適用範囲

区分内容
対象事象大規模地震（震度6弱以上）、水害（床上浸水・床下浸水）、大規模停電・通信障害、新型インフルエンザ等感染症の蔓延、電子カルテのダウン・ランサムウェア等サイバー攻撃
対象者医療法人いなだ医院の全職員（常勤・非常勤含む）
施設いなだ医院（鉄筋コンクリート造）

1.4　発動基準

種別発動の目安
自然災害版震度6弱以上の地震発生時、または医院が浸水リスク区域に被害情報が出た時点
感染症版国・県が「警戒宣言」または「緊急事態宣言」を発出した時点、または院内で感染クラスターが発生した時点
サイバーセキュリティ版電子カルテが使用不能となった時点、またはサイバー攻撃の疑いが生じた時点

第2章　推進体制（指揮・連絡系統）

2.1　対策本部の設置

緊急事態発生時、院長を本部長とする「災害（緊急対策）本部」を設置する。本部は原則として院長室に置く。建物が使用不能な場合は駐車場を代替本部とする。

第4章　自然災害版　初動対応

4.1　大規模地震発生時

①　発災直後（〜30分）

  1. 診察中の場合は机の下・患者を壁際に誘導し、揺れが収まるまで待機する
  2. 揺れ収束後、院内巡回（火災・ガス漏れ・構造損傷・患者負傷）を即座に確認する
  3. 火災・ガス漏れを確認した場合は119番通報→全員避難を最優先とする
  4. 院長（本部長）が「診療継続 / 休診」を宣言する

②　停電時の対応

  1. 備蓄ポータブル蓄電器を起動し、電子カルテ端末・最低限の照明の電源を確保する
  2. 電力会社（中国電力：0120-266-995）へ停電状況を報告する

③　継続可否の判断・告知

  1. 継続困難な場合は「休診」を決定し、入口ボードに掲示する（様式は第8章参照）
  2. 玉島医師会に被災・診療状況を報告する
  3. 訪問診療患者へ電話で方針を伝達する

4.2　水害発生時

  1. 洪水ハザードマップで医院周辺の浸水予測を確認し、行政の避難指示に従う
  2. 浸水前に電子カルテ・医療機器の高所移動を行い、電源を切断する
  3. 避難が必要な場合は院内患者を安全に避難誘導し、訪問診療患者へ連絡する
  4. 浸水後は電気系統の安全確認（専門業者による確認）が取れるまで電源を入れない

4.3　被害確認チェックリスト

発災後、各担当者が以下を確認し、対策本部に報告する（様式1を使用）。

第5章　感染症版　初動対応

5.1　発動基準

状況対応レベル
院内で感染症クラスター発生（スタッフ陽性者2名以上）レベル2：制限診療（優先度【低】を即停止）
国・県から「緊急事態宣言」発出レベル3：最低限診療（優先度【高】のみ継続）
出勤可能スタッフが通常の50%以下レベル3：最低限診療に移行

5.2　動線分離と感染対策

  • 発熱外来と一般・小児科外来を空間的・時間的に完全分離する（午前/午後で使い分けるか別室使用）
  • 全スタッフは適切なPPE（マスク・ガウン・手袋・フェイスシールド）を着用する
  • 発熱患者は院内待機ではなく自家用車待機を原則とする
  • 待合室の密度管理（座席間隔1m以上・人数制限）を行う

5.3　スタッフ管理

状況対応
スタッフが陽性確定即日自宅療養。復帰基準は発症から10日間経過かつ症状消失後72時間以上
スタッフが濃厚接触者原則待機（院長判断でPPE着用条件付き勤務可）。毎日抗原検査を実施
スタッフ不足（50%以下）リハビリ等を即停止し、優先度【高】業務に集中。外部人材派遣・近隣医院への応援依頼を検討

5.4　診療体制の変更

  • 定期健康診断・予防接種（感染症対応のものを除く）は中止し、患者に連絡を入れる
  • 慢性疾患患者への処方は電話診療・オンライン診療への切替を検討する
  • 電話診療を実施する場合は、玉島医師会への連絡・所定の届出を行う

第6章　サイバーセキュリティ版　初動対応

6.1　電子カルテダウン時（紙運用への切替）

【切替手順】

  1. 電子カルテ端末が起動しない・操作できない場合、即座に紙カルテ運用に切り替える
  2. 「システムダウン時用キット」を取り出す（保管場所：院内倉庫）
    内容：紙カルテ用紙・手書き処方箋・問診票・会計伝票・患者名簿（直近3か月分）・印鑑
  3. 患者からの聞き取りとお薬手帳を最大限活用し、処方履歴を確認する
  4. 電子カルテ保守ベンダー（ユヤマ：0120-610-134）へ緊急連絡を行い、復旧見込みを確認する
  5. 復旧後、紙カルテの内容を電子カルテへ遡及入力する

6.2　ランサムウェア・サイバー攻撃疑い時

【絶対にやること】

  1. 感染が疑われる端末を即座にネットワークから切断する（LANケーブルを抜く・Wi-Fiをオフ）
  2. 院内すべての端末のインターネット接続を遮断する（ルーターの電源をオフ）
  3. ユヤマ緊急連絡先へ連絡し、バックアップからのリストアを依頼する
  4. 倉敷市保健所・岡山県医師会へ報告する
  5. 身代金の要求には応じない

【やってはいけないこと】

  • 感染が疑われる端末でファイルを開いたり保存したりする
  • ウイルス対策ソフトを無効化する
  • 独断でバックアップから復元を試みる
